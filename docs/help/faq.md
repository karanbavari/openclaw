# FAQ

This fork is a web-first SaaS build focused on gateway, agents, channels, providers, and the web admin UI.

## What was removed?

- CLI-first onboarding
- local install runbooks
- browser automation docs
- web search docs
- terminal UI docs
- native node/device app docs

## What is supported now?

- Web admin/control UI
- Gateway APIs
- Internal multi-agent runtime
- Retained messaging channels and model providers configured for this fork

## Setup path

Use the web admin setup flow for tenant, channel, provider, and workspace configuration.
